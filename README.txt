📦 QTY Network License - Installation Instructions

Thank you for purchasing the QTY Network app. Please follow these steps to activate your license on your computer:

──────────────────────────────────────────────
STEP1️⃣: Run the Request Tool
──────────────────────────────────────────────

1. Double-click `request_tool.exe`
2. It will automatically send your Machine ID to the license server.
3. If your license has been approved, it will also download and run the installer.
4. If approval is pending, you'll see a message:  
   👉 "Installer not yet approved. Please run this tool again later."

   🔁 You may re-run the tool after some time to check again.

──────────────────────────────────────────────
STEP 2️⃣: License Approval (Admin Action)
──────────────────────────────────────────────

⚠️ IMPORTANT:
You do NOT need to send your machine ID manually.
It is sent automatically to the server for approval.

After approval, the tool will download the licensed file.

──────────────────────────────────────────────
STEP 3️⃣: Open the Launcher
──────────────────────────────────────────────

1. Open `Launcher.xlsm`
2. It will automatically open the licensed file:  
   ➤ `QTY_Network_2025_<YourMachineID>.xlsm`
3. If the license is not valid or not found, Excel will auto-close.

💡 Do not rename the licensed file. Keep the full name as downloaded.

──────────────────────────────────────────────
⚠️ Notes
──────────────────────────────────────────────

- This app is locked to your computer’s Machine ID.
- The license is valid **only on the machine where it was approved**.
- If you disable macros, the tool will not run properly.
- You may keep a backup of your licensed `.xlsm` file, but it will only open on your approved machine.

──────────────────────────────────────────────

Thank you again!
